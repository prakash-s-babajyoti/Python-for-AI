## 📚 Curriculum Breakdown

### 1. Environment Setup
* Python installation & version management (pyenv)
* VS Code / PyCharm setup
* Virtual environments (venv, conda)
* pip & package management
* Git & GitHub basics

### 2. Core Language
* Data types: `int`, `float`, `str`, `bool`, `None`
* Variables, operators, expressions
* Control flow: `if`/`elif`/`else`, loops (`for`, `while`)
* Functions: `args`, `kwargs`, default params
* Scope, closures, lambda functions

### 3. Data Structures
* Lists, tuples, sets, dicts
* List & dict comprehensions
* Slicing, indexing, unpacking
* Built-in functions: `map`, `filter`, `zip`, `enumerate`
* Nested structures & iteration

### 4. OOP Basics (Object-Oriented Programming)
* Classes & objects
* Constructors (`__init__`)
* Instance vs class variables
* Inheritance & polymorphism
* Dunder/magic methods

### 5. Error Handling & Files
* `try`/`except`/`finally` blocks
* Custom exceptions
* File I/O (`open`, `read`, `write`)
* Context managers (`with` statement)
* JSON & CSV handling
